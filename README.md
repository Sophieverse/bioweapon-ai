# bioweapon.ai

A public awareness website about AI-enabled biosecurity risks, based on the research document "Mapping Emergent AI-Aided Biosecurity Risks."

## About

This site presents publicly available research to raise awareness about how artificial intelligence is eliminating the technical barriers that have historically prevented bioterrorism. All data is sourced from published academic literature, government reports, and public statements by AI company executives.

## Deployment

Static HTML site — deploy to any web host. For Hostinger, upload `index.html` to the public_html folder or connect this GitHub repo via Hostinger's Git deployment feature.
